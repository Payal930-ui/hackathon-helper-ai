# Hackathon Helper AI

A premium AI-powered SaaS application that helps students plan, build, and pitch hackathon projects. Enter your idea and get project plans, tech stacks, database schemas, UI/UX suggestions, code snippets, PPT presentations, README files, and deployment guides — all generated by Google Gemini.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15, TypeScript, Tailwind CSS, Framer Motion |
| Backend | Next.js API Routes |
| Database | Firebase Firestore |
| Auth | Firebase Authentication |
| AI | Google Gemini API |
| PPT | pptxgenjs |
| Deployment | Vercel |

## Features

- **Authentication**: Sign up, login, logout, forgot password, protected routes
- **Theme**: Light/dark mode with Firestore persistence
- **Project Creation**: 3-step wizard with selectable AI outputs
- **History**: Search, view, edit, delete, and regenerate projects
- **Exports**: Copy to clipboard, PDF, README download, PPT download
- **Dashboard**: Modern sidebar navigation with glassmorphism UI

## Database Schema

### Users Collection (`users/{uid}`)

| Field | Type | Description |
|-------|------|-------------|
| uid | string | Firebase Auth UID |
| name | string | Display name |
| email | string | Email address |
| theme | string | `"light"` or `"dark"` |
| createdAt | string | ISO timestamp |

### Projects Collection (`projects/{projectId}`)

| Field | Type | Description |
|-------|------|-------------|
| userId | string | Owner's Firebase UID |
| title | string | Project title |
| description | string | Project description |
| selectedOutputs | string[] | Selected output keys |
| generatedResults | object | AI-generated content |
| createdAt | string | ISO timestamp |
| updatedAt | string | ISO timestamp |

## Getting Started

### 1. Clone and Install

```bash
npm install
```

### 2. Environment Variables

Copy `.env.local.example` to `.env.local` and fill in your credentials:

```bash
cp .env.local.example .env.local
```

### 3. Firebase Setup

1. Create a project at [Firebase Console](https://console.firebase.google.com)
2. Enable **Email/Password Auth** and **Cloud Firestore**
3. Create a **Composite Index** for queries to work:
   - Collection: `projects`
   - Fields: `userId` (Ascending), `createdAt` (Descending)
   - Scope: Collection
   - *Alternatively, run the `History` page and click the link in the console error to automatically create it.*
4. Copy config values to `.env.local`

### 4. Gemini API Key

Get your key from [Google AI Studio](https://aistudio.google.com/apikey) and add it as `GEMINI_API_KEY`.

### 5. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## API Routes

### `POST /api/generate`

Generates AI content for a project.

**Request:**
```json
{
  "title": "EcoTracker AI",
  "description": "An app that tracks carbon footprint...",
  "selectedOutputs": ["projectPlan", "techStack", "readme"]
}
```

**Response:**
```json
{
  "generatedResults": {
    "projectPlan": "...",
    "techStack": { "frontend": "...", "backend": "..." }
  }
}
```
